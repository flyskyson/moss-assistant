# MOSS Assistant - 启动说明

## 快速启动（3 种方式）

### 方式 1: 桌面快捷方式（推荐）✅
- 在桌面找到 **MOSS Assistant** 图标
- 双击运行
- 浏览器会自动打开 http://localhost:8501

### 方式 2: StartMOSS.bat
- 双击项目目录中的 `StartMOSS.bat`
- 浏览器会自动打开 http://localhost:8501

### 方式 3: Python 启动器
```bash
cd C:\Users\flyskyson\moss-assistant
python launcher.py
```

---

## 文件说明

| 文件 | 说明 |
|------|------|
| **StartMOSS.bat** | 简单启动脚本（推荐） |
| **快速启动.bat** | 带检查的启动脚本 |
| **一键启动.bat** | 完整检查的启动脚本 |
| **launcher.py** | Python 智能启动器 |
| **CreateShortcut.ps1** | 创建桌面快捷方式 |

---

## 停止服务

- 在命令行窗口按 **Ctrl+C**
- 或直接关闭命令行窗口

---

## 故障排查

### 问题 1: Python 未安装
**解决**: 下载安装 Python 3.8+
https://www.python.org/downloads/

### 问题 2: 端口被占用
**解决**:
```bash
# 查找占用进程
netstat -ano | findstr :8501

# 结束进程
taskkill /F /PID <进程ID>
```

### 问题 3: 依赖缺失
**解决**:
```bash
pip install -r requirements.txt
```

如果桌面快捷方式丢失，运行：

或右键 `CreateShortcut.ps1` → 使用 PowerShell 运行
