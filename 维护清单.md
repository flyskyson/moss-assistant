## 常用命令

### 启动
```bash
cd C:\Users\flyskyson\moss-assistant
python start_moss.py
```

### 停止
```bash
taskkill //F //IM python.exe
```

### 查看日志
```bash
tail -100 streamlit.log
```

### Git 操作
```bash
# 查看状态
git status

# 提交更改
git add .
git commit -m "描述"
git push

# 更新代码
git pull
```

## 文件位置
- 配置文件: `config.yaml`
- 核心代码: `core/`
- Web 界面: `app.py`
- 数据存储: `data/`
- 文档: `*.md`

## 常见问题
- 启动失败 → 检查 streamlit.log
- 记忆丢失 → 恢复备份
- API 错误 → 检查 .env 文件
- 端口占用 → 换个端口
