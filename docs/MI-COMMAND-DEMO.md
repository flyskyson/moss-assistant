# mi 命令 - 完整演示

**日期**: 2026-02-06
**命令**: `mi`

---

## 🎬 完整使用流程演示

### 步骤 1: 打开终端

在 macOS 上打开 **Terminal** 或 **iTerm2**：

```
位置:
- 应用程序 → 实用工具 → 终端
- 或 Spotlight 搜索 "Terminal"
```

---

### 步骤 2: 输入命令

在终端中输入：

```bash
mi
```

**按回车键**

---

### 步骤 3: 查看确认信息

终端会显示：

```
✅ 初始化提示已复制到剪贴板！
📝 现在可以直接粘贴给 MOSS
```

---

### 步骤 4: 切换到 MOSS

打开 MOSS 对话窗口

---

### 步骤 5: 粘贴

按 **Cmd + V**（同时按 Command 键和 V 键）

---

## 🔍 验证命令是否可用

### 测试 1: 检查 alias

```bash
$ alias mi
mi='~/clawd/scripts/moss-init.sh'
```

**如果显示这个** → ✅ 配置正确

**如果显示这个**:
```
zsh: command not found: mi
```

**解决**:
```bash
source ~/.zshrc
```
