## 🔧 故障排除

### 问题 1: MOSS 说不读取文件

**原因**: 可能是旧会话缓存

**解决**:
```bash
rm -rf ~/.openclaw/agents/main/sessions/
openclaw gateway restart
```

---

### 问题 2: mi 命令不存在

**原因**: alias 未生效

**解决**:
```bash
source ~/.zshrc
```
