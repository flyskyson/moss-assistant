## 🔥 推荐启动方式（按优先级）

### 🥇 首选：一键启动脚本

**Windows:**
```bash
# 双击运行
一键启动.bat
```

**Linux/Mac:**
```bash
./一键启动.sh
```

**优点:**
- ✅ 完整的 6-7 步启动流程
- ✅ 自动环境检测
- ✅ 自动依赖安装
- ✅ 健康检查
- ✅ 美观的界面
- ✅ 详细的错误提示

### 🥈 次选：智能启动器

```bash
python launcher.py
```

**优点:**
- ✅ Python 原生，更可靠
- ✅ 可定制性强
- ✅ 跨平台统一
- ✅ 适合集成

### 🥉 备选：快速启动

**Windows:**
```bash
启动 MOSS.bat
```

**优点:**
- ✅ 最快
- ✅ 最简洁
- ✅ 适合频繁使用

## 🛠️ 故障排查快速参考

| 问题 | 解决方案 |
|------|---------|
| Python 未安装 | 下载 Python 3.8+ |
| 依赖安装失败 | `pip install -r requirements.txt` |
| 端口被占用 | 关闭占用 8501 端口的进程 |
| API Key 未设置 | 创建 `.env` 文件并添加 `DEEPSEEK_API_KEY` |
| 编码错误 (Windows) | 运行 `chcp 65001` |

详细故障排查请查看 [启动指南](./启动指南.md)
