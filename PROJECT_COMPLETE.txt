🚀 立即开始
────────────────────────────────────────────────────────────────

# 1. 安装依赖
pip install -r requirements.txt

# 2. 设置 API Key
export ANTHROPIC_API_KEY="你的密钥"

# 3. 启动
streamlit run app.py

然后打开: http://localhost:8501
