## 📋 启动流程

### 完整的 7 步启动流程

```
[1/7] 检测运行环境
  └─ Python 版本检查
  └─ pip 可用性检查

[2/7] 加载环境变量
  └─ 读取 .env 文件
  └─ 验证 DEEPSEEK_API_KEY

[3/7] 检查项目依赖
  └─ 检查 requirements.txt
  └─ 自动安装缺失的包

[4/7] 检查核心文件
  └─ 验证 moss.py、app.py、config.yaml
  └─ 验证 core/ 目录

[5/7] 检查数据目录
  └─ 创建 data/ 目录
  └─ 创建 data/logs/ 目录

[6/7] 健康检查
  └─ 验证配置文件
  └─ 检查端口 8501

[7/7] 启动服务
  └─ 启动 Streamlit
  └─ 自动打开浏览器
```

## 🛠️ 故障排查

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| Python 未安装 | 下载 Python 3.8+ |
| 依赖安装失败 | `pip install -r requirements.txt` |
| 端口被占用 | 关闭占用 8501 端口的进程 |
| API Key 未设置 | 创建 `.env` 文件并添加 `DEEPSEEK_API_KEY` |
| 编码错误 (Windows) | 运行 `chcp 65001` |

**详细故障排查**: 查看 [启动指南.md](./启动指南.md)
