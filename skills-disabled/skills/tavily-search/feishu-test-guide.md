## 常见问题排查

### 问题 1: MOSS 说没有搜索功能
**原因**: MOSS 没有读取更新后的 AGENTS.md

**解决**:
```bash
# 检查 AGENTS.md 是否包含搜索说明
grep "Web Search" /Users/lijian/clawd/AGENTS.md

# 重启 gateway
openclaw gateway restart
```

### 问题 2: 搜索结果不准确
**原因**: 查询语句不够明确

**解决**: 教 MOSS 如何构造更好的搜索查询
- 使用关键词而不是整句
- 添加年份限制（如 "2026"）
- 使用英文搜索技术文档

**检查**:
