# 🚀 MOSS Assistant - 快速参考卡片

## ⚡ 一键启动

### 最简单的方式 ✨
```
双击桌面上的 "MOSS Assistant" 图标
```

### 其他方式
```bash
# 方式 1: StartMOSS.bat
双击项目目录中的 StartMOSS.bat

# 方式 2: Python 启动器
python launcher.py

# 方式 3: 命令行
python -m streamlit run app.py
```

---

## 🌐 访问地址

```
http://localhost:8501
```

## 🛑 停止服务

```
按 Ctrl+C
或关闭命令行窗口
```

## 🔧 常见问题

### Python 未安装
→ 下载: https://www.python.org/downloads/

### 端口被占用
```bash
netstat -ano | findstr :8501
taskkill /F /PID <进程ID>
```

### 依赖缺失
```bash
pip install -r requirements.txt
```
