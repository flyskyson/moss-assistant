## 🛠️ 故障排查

### Python 未安装
```
❌ 错误: 未检测到 Python

💡 解决:
1. 访问 https://www.python.org/downloads/
2. 下载并安装 Python 3.8+
3. 重新运行启动脚本
```

### 端口被占用
```
❌ 错误: 端口 8501 已被占用

💡 解决:
# Windows
netstat -ano | findstr :8501
taskkill /F /PID <进程ID>

# 或者在启动时指定其他端口
streamlit run app.py --server.port 8502
```

### 依赖缺失
```
❌ 错误: ModuleNotFoundError

💡 解决:
pip install -r requirements.txt
```
